# RxJS Time Module #

The Reactive Extensions for JavaScript has a built-in mechanism for testing all operators which allows for mocking absolute and relative time with ease.

## Details ##

Files:
- `rx.testing.js`

NPM Packages:
- `rx`

NuGet Packages:
- `RxJS-Time`

File Dependencies:
- `rx.js` | `rx.compat.js` | `rx.lite.js` | `rx.lite.compat.js`
- `rx.virtualtime.js`

NPM Dependencies:
- None

NuGet Dependencies:
- RxJS-Main
- RxJS-VirtualTime
